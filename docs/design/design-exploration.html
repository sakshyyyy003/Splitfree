Layout rules

  • 12-column desktop feel, but cards stay rounded and tactile.
  • Primary CTA is solid moss, secondary action is outlined on white.

  • Forms stay single-column until there is room to compare related inputs.

Flow note

Invite options should feel optional, not mandatory

Users can create the group first and add people later. The UI should never make invite completion feel like a blocker.

Field treatment
